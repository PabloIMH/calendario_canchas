# 🏟️ Canchas Reservas

Sistema profesional de reservas de canchas deportivas con tecnología PWA.

## ✨ Características

- 📱 **PWA Instalable** - Instala como app nativa
- 🇨🇱 **Zona Horaria Chilena** - Siempre precisa
- 🔄 **Reservas Permanentes** - Recurrencia semanal
- 🎨 **Diseño Premium** - Glassmorphism moderno
- 📊 **Dashboard Admin** - Gestión completa
- 🌐 **Offline Ready** - Funciona sin internet

## 🚀 Instalación

### Como PWA
1. Abre la aplicación en Chrome
2. Espera 3 segundos
3. Toca "Instalar App"
4. Listo 🎉

### Como Web App
1. Ve a [GitHub Pages](https://pabloimh.github.io/calendario_canchas/)
2. Usa directamente en el navegador
3. Sin instalación requerida

## 🔧 Tecnologías

- **Frontend**: HTML5, CSS3, JavaScript ES6+
- **Backend**: Firebase Firestore
- **PWA**: Service Worker + Web App Manifest
- **Auth**: Firebase Authentication
- **Deploy**: GitHub Pages

## ⚙️ Configuración

1. Clona el repositorio:
```bash
git clone https://github.com/PabloIMH/calendario_canchas.git
```

2. Configura Firebase:
- Crea proyecto en [Firebase Console](https://console.firebase.google.com/)
- Habilita Firestore y Authentication
- Copia las credenciales en `index.html`

3. Deploy automático:
- Push a `main` → Deploy automático a GitHub Pages
